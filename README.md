## AMInventory 1.0.0
Models, functions and algorithms for Inventory Control and Optimization, largely  based on material from Chapters 3-7 of the book ""Axs√§ter, S. (2015) Inventory Control, 3d Edition, Springer". Purpose to provide a class companion to the mentioned book without requiring any deeper knowledge of R or any programming skills.

Install by issuing in R:

remotes::install_github("athmig59/AMInventory", ref="main")

List contents and use by issuing in R:

library(AMInventory)

ls("package:AMInventory")

Help and examples are available for all funktions. Check, for instance, the "economic order quantity" function by issuing:

?EOQ
